## 🏢 Convention

| Tag Type   | content                       |
|------------|-------------------------------|
| `Feat`     | 기능(feature)                   |
| `Docs`     | 문서 작업 (documentation)         |
| `Fix`      | 버그 수정                         |
| `Hotfix`   | 긴급 버그 수정                      |
| `Refactor` | 코드 리팩터링(가독성 향상)               |
| `Ci`       | ci 파이프라인 구축 관련                |
| `Ci/Cd `   | ci 빌드 수정 및 deployment         |
| `Chore`    | 관리(maintain), 핵심 내용은 아닌 기타 변경 |
